Post-hoc Compression for 3D Gaussian Splatting

Runner-up · COGS’25 (ICCV 2025) Model-agnostic Pruning + QAT + Entropy Coding

We present a post-hoc compression pipeline for pretrained 3D Gaussian Splatting (3DGS) models. The method alternates gradient accumulation and gradient-informed pruning with quantization-aware fine-tuning, followed by entropy coding. It reduces model size and improves frame rate with minimal loss in visual quality.

Key Numbers

~30k total fine-tuning steps
5% pruning per iteration
1 pass gradient accumulation

Pipeline

  1. Accumulate gradients on the training set (one pass).
  2. Prune Gaussians with the smallest gradient magnitudes.
  3. Fine-tune with quantization awareness (SH→uint8, others→FP16).
  4. Repeat until the stopping criterion is met; finalize with entropy coding (LZMA).

Objective: reduce memory footprint while preserving PSNR/SSIM and sustaining high FPS.

Method Details

  • Spherical harmonics (SH) coefficients are quantized to uint8; other parameters are stored in FP16.
  • We apply an L1 penalty on SH coefficients to encourage sparsity.
  • Pruning proceeds in 5% steps by ascending gradient magnitude; quality is checked after each iteration.
  • We stop when PSNR ≤ baseline − 0.1 dB. Morton ordering is applied prior to entropy coding (LZMA).
